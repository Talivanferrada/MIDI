# PROJECT — MIDI Framework

## Name
**MIDI — Modelo Inteligente de Desarrollo de Innovación**

## Definition
MIDI es un framework multiagente instalable que permite transformar una idea inicial de negocio, emprendimiento, innovación o proyecto en un documento maestro completo, analizado, tensionado, validado y adaptable a fondos concursables, inversionistas o ejecución real.

## Vision
Convertirse en una capa de trabajo instalable para transformar ideas iniciales en proyectos reales, financiables y ejecutables mediante archivos, agentes, skills, comandos y workflows que puedan ser usados por distintos agentes de desarrollo.

## Core Principle
MIDI no debe enamorarse de la idea inicial. Debe investigar, ampliar, comparar, destruir, mejorar, validar y recién después estructurar.

## Geographic Focus
Enfoque global con adaptación a Chile. Debe poder aterrizar cualquier idea al contexto chileno cuando el usuario lo requiera.

## Two Modes

### Mode 1: Exploration
- Objetivo: ampliar posibilidades, investigar referentes, detectar tendencias, generar ideas y seleccionar las mejores.
- Produce: investigación, benchmark, insights, 10-15 ideas, top 3.

### Mode 2: Financeable
- Objetivo: tomar una idea seleccionada y convertirla en un proyecto ejecutable, defendible y financiable.
- Produce: análisis profundo, BMC, Lean Canvas, factibilidad, riesgos, MVP, validación, documento final.
