# Skill: market-analysis

## Description
Investigación y análisis de mercado para determinar tamaño, tendencias, segmentación y dinámicas competitivas de un sector específico.

## When to use
- En Modo Financiable para validar tamaño de oportunidad
- Cuando se necesita data para inversionistas
- Para entender dinámicas de un sector específico
- Al evaluar la viabilidad comercial de una idea
- Para identificar segmentos atractivos dentro de un mercado

## Process
1. **Definición del Mercado**
   - Delimitar el mercado objetivo
   - Identificar mercado total, direccionable y obtenible (TAM/SAM/SOM)
   - Clarificar scope geográfico

2. **Investigación Cuantitativa**
   - Buscar datos de tamaño de mercado
   - Identificar tasas de crecimiento
   - Revisar reportes de industria
   - Validar con múltiples fuentes

3. **Análisis Cualitativo**
   - Identificar tendencias clave
   - Analizar dinámicas competitivas
   - Evaluar barreras de entrada
   - Detectar cambios regulatorios

4. **Segmentación**
   - Dividir mercado en segmentos significativos
   - Evaluar atractividad de cada segmento
   - Priorizar segmentos objetivo

5. **Síntesis y Proyecciones**
   - Consolidar hallazgos
   - Proyectar posición de mercado esperada
   - Identificar riesgos y oportunidades

## Examples

### Ejemplo de uso
```
Usuario: Necesito entender el mercado de delivery de comida en Chile.

Agente con skill market-analysis:
1. TAM: Mercado food service Chile USD 8B
2. SAM: Delivery food Chile USD 800M (10% penetration)
3. SOM: Target 2% en 3 años = USD 16M

Tendencias:
- Crecimiento 25% anual post-pandemia
- Saludable = segmento creciendo al 35%
- Expectativa de <30 min delivery

Competencia:
- Uber Eats 40%, PedidosYa 35%, Rappi 25%
- Ninguno tiene foco exclusivo en saludable

Segmentos:
- Premium: dispuestos a pagar +30% por saludable
- Conveniencia: valoran velocidad sobre variedad
- Planificados: suscripciones semanales
```
