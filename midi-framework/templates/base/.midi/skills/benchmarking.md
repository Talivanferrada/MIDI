# Skill: Benchmarking

## Description
Análisis comparativo de competidores, referentes y casos de éxito para extraer patrones aplicables al proyecto.

## When to use
- Cuando se necesita entender el panorama competitivo
- Para identificar mejores prácticas del mercado
- Antes de definir diferenciación
- Cuando se evalúan modelos de negocio similares

## Process
1. **Identificar competidores**
   - Definir criterios de búsqueda
   - Listar competidores directos e indirectos
   - Identificar referentes internacionales

2. **Análisis comparativo**
   - Evaluar propuestas de valor
   - Comparar modelos de negocio
   - Analizar estrategias de precios
   - Identificar fortalezas y debilidades

3. **Extracción de patrones**
   - Identificar elementos exitosos
   - Detectar gaps del mercado
   - Encontrar oportunidades de diferenciación

4. **Documentación**
   - Crear matriz comparativa
   - Generar recomendaciones
   - Actualizar `01_research/benchmark.md`

## Templates
- `templates/market-analysis.md`
- `templates/top3-comparison.md`

## Outputs
- `01_research/benchmark.md` - Análisis de referentes
- Matriz comparativa de competidores
- Recomendaciones de diferenciación

## Examples

### Ejemplo: Benchmark de apps de delivery
```
User: Analizar competidores de delivery de comida
Agent: Ejecutaré benchmarking para:
1. Identificar top 5 apps de delivery en Chile
2. Analizar sus modelos de negocio y precios
3. Comparar propuestas de valor
4. Identificar oportunidades de diferenciación
5. Generar documento de benchmark completo
```
