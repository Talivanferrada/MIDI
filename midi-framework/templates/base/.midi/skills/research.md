# Skill: Research

## Description
Metodología de investigación global y local para descubrir oportunidades, tendencias y referentes relevantes para el proyecto.

## When to use
- Al inicio de cualquier proyecto para establecer contexto
- Antes de generar ideas para descubrir oportunidades
- Cuando se necesita validar supuestos con datos
- Para encontrar referentes y benchmarks del mercado

## Process
1. **Definir alcance de investigación**
   - Identificar el área de interés
   - Determinar tipo de investigación (global vs local)
   - Establecer fuentes relevantes

2. **Investigación global**
   - Buscar tendencias internacionales
   - Identificar innovaciones y disruptores
   - Analizar modelos de negocio exitosos
   - Documentar hallazgos en `01_research/global_research.md`

3. **Investigación local**
   - Adaptar hallazgos al contexto chileno/latinoamericano
   - Identificar actores locales relevantes
   - Analizar regulaciones y condiciones del mercado
   - Documentar en `01_research/local_adaptation.md`

4. **Síntesis**
   - Consolidar hallazgos principales
   - Identificar gaps y oportunidades
   - Generar insights accionables
   - Actualizar `DECISION_LOG.md`

## Templates
- `templates/market-analysis.md`
- `templates/intake-form.md`

## Outputs
- `01_research/global_research.md` - Investigación global
- `01_research/local_adaptation.md` - Adaptación local
- `01_research/benchmark.md` - Análisis de referentes
- `02_insights/insights.md` - Insights generados

## Examples

### Ejemplo: Investigación de mercado de alimentos
```
User: Necesito investigar el mercado de snacks saludables
Agent: Ejecutaré la skill de research para:
1. Investigar tendencias globales en snacks saludables
2. Analizar mercado chileno de alimentos saludables
3. Identificar competidores y referentes internacionales
4. Generar documento de research completo
```
