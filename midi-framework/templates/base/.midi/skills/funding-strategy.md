# Skill: funding-strategy

## Description
Identificación y análisis de fuentes de financiamiento disponibles para proyectos en Chile, incluyendo fondos concursables, inversionistas, créditos y programas de apoyo.

## When to use
- En Modo Financiable para definir estrategia de capital
- Cuando el proyecto requiere financiamiento externo
- Para identificar fondos concursables aplicables
- Al preparar pitch para inversionistas
- Para evaluar opciones de financiamiento

## Process
1. **Diagnóstico de Necesidades**
   - Calcular capital requerido
   - Definir uso de fondos
   - Establecer timeline de levantamiento
   - Clarificar stage del proyecto

2. **Mapeo de Fuentes**
   - Identificar fondos concursables CORFO
   - Revisar programas STARTUP CHILE
   - Buscar inversionistas ángeles
   - Evaluar VC relevantes
   - Considerar créditos y líneas bancarias

3. **Análisis de Elegibilidad**
   - Revisar requisitos de cada fondo
   - Evaluar fit con perfil del proyecto
   - Identificar gaps de cumplimiento
   - Priorizar fuentes más viables

4. **Estrategia de Abordaje**
   - Definir secuencia de acercamiento
   - Preparar materiales específicos por fuente
   - Establecer contactos clave
   - Planificar timeline de postulación

5. **Plan B y Alternativas**
   - Identificar opciones si se rechaza
   - Considerar bootstrapping
   - Evaluar partnerships estratégicos

```
Usuario: Necesito financiar mi proyecto de fintech para pymes.

Agente con skill funding-strategy:
1. Necesidad: $150M CLP para MVP + 12 meses

2. Fuentes identificadas:
   - CORFO Semilla Inicia: hasta $40M (70% subsidio)
   - STARTUP CHILE: $40M + network
   - Inversionistas ángeles: $70M por 15%
   - CORFO Línea Crédito: $50M a tasa subsidiada

3. Elegibilidad:
   - Semilla Inicia: ✓ Team completo, ✓ Innovación, ✓ Sin ventas
   - STARTUP CHILE: ✓ Stage early, ✓ Scalable, ✓ Tech
   - Ángel: ✓ Tracción necesaria, ✗ Todavía no hay

4. Recomendación:
   - Fase 1: Postular Semilla Inicia (deadline en 2 meses)
   - Fase 2: Aplicar STARTUP CHILE (batch siguiente)
   - Fase 3: Levantar ángel con tracción del pilot

5. Timeline:
   - Mes 1-2: Preparar postulación Semilla
   - Mes 3-6: Ejecutar con fondos propios + bootstrapping
   - Mes 6-9: STARTUP CHILE + levantamiento ángel
```

- CORFO Innova: Hasta $100M CLP (Innovación I+D)
